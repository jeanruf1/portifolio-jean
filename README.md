## English

A highly responsive, modern, and bilingual portfolio designed to showcase projects, experience, and expertise in Data Engineering and Artificial Intelligence. Built using Next.js 14, Tailwind CSS, and Framer Motion.

### Features
* **Bilingual Architecture:** Instant client-side toggle between English and Portuguese.
* **Modern Interface:** Smooth micro-interactions and transitions powered by Framer Motion.
* **Direct Form Delivery:** Contact form powered directly by Web3Forms on the client side.
* **Docker Ready:** Complete support for Docker and Docker Compose for simple, isolated development.
* **Hydration Protection:** Pre-configured with static data imports to prevent SSR mismatch errors.

### Tech Stack
| Tool | Scope |
| :--- | :--- |
| Next.js 14 | React framework with App Router for fast, optimized routing |
| Tailwind CSS | Utility-first framework for responsive and custom layout styling |
| Framer Motion | Smooth, declarative animations and gesture animations |
| Web3Forms | Client-side email dispatch integration |
| Docker | Containerization for reproducible local environments |

### 2. Configure Environment Variables
Copy the `.env.example` file to create your local `.env.local` configuration.
```bash
cp .env.example .env.local
```
Add your free Web3Forms access key to `.env.local` to enable contact form submissions.

### 3. Run with Docker (Recommended)
This project is configured to run flawlessly with Docker and Docker Compose.
```bash
docker compose up
```
Open `http://localhost:3000` to view it in the browser.

### 4. Or run locally with Node.js
If you prefer running directly on your host machine:
```bash
npm install
npm run dev
```
